Instalación de un Sistema Acuapónico

Para poder realizar un sistema acuaponico debes de saber como se usa, a continuacion te explicaremos como:

Un sistema acuaponico se usa para criar peces y cultivar plantas en un mismo espacio, de manera que los desechos de los peces fertilicen las plantas y las raices de las plantas purifiquen el agua.

PARA USAR UN SISTEMA ACUAPONICO, SE DEBE:

  • Asegurar que el agua este en buen estado.
  • Evitar sobrealimentar a los peces.
  • Elegir y espaciar las plantas con cuidado.
  • Mantener un equilibrio entre las plantas y los peces.
  • Filtrar los solidos y realizar biofiltracion.

COMPONENTES DEL SISTEMA ACUAPONICO

  • Tanque de peces.
  • Sistema de filtracion.
  • Biofiltro.
  • Sistema de cultivo de plantas.
  • Sistema de bombeo.

PASOS PARA CREAR UN SISTEMA ACUAPONICO

  • Diseñar el sistema:Determinar el tamaño y la configuracion del sistema acuaponico, incluyendo el tamaño del tanque de peces y el sistema de cultivo de plantas.
  • Instalar el tanque de peces:Instalar el tanque de peces y agregar los peces y otros organismops acuaticos.
  • Instalar el sistema de filtracion:Instalar el sistema de filtracion para eliminar los desechos y mantener la calidad del agua.
  • Instalar el biofiltro:Instalar el biofiltro para convertir los desechos en nutrientes para las plantas.
  • Instalar el sistema de cultivo de plantas:Instalar el sistema de cultivo de plantas,
    como un sistema de flujo continuo o un sistema de goteo.
  • Conectar el sistema de bombeo:Conectar el sistema de bombeo para bombear el agua
    desde el tanque de peces hasta el sistema de cultivo de plantas.
  • Monitorear y mantener el sistema:Monitorear y mantener el sistema acuaponico para asegurarse de que este funcionando correctamente y que las plantas y que los peces esten saludables.
